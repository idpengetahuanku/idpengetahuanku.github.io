10. Jika kartu SD terenkripsi apakah file yg ada di kartu akan hilang ?


Jawaban:

Ide dari enkripsi SD card ini memang untuk keamanan data. Smartphone hilang atau dicuri, biasanya smartphonenya sendiri terlindung penggunaan password atau kunci biometrik, tanpa ini, data di internal storage kita relatif aman untuk tidak dilihat atau digunakan pencuri.

Tetapi data di SD card bisa dengan mudah dibaca lewat perangkat lain, misal card reader atau smartphone lain. Untuk itu demi keamanan data menyeluruh, diberikan fitur untuk enkripsi SD card, sehingga ketika

SD card dicoba dibaca di perangkat lain, tidak akan terbaca.

Masalahnya ada 2 kunci di enkripsi SD card ini, satu kunci hardware, berupa data perangkat yang digunakan saat enkripsi (smartphone atau tablet), dan satu kunci password (atau pattern atau pin) untuk unlock smartphone.

Kedua kunci ini tidak di sync ke cloud atau kemanapun demi keamanan, dan disimpan di dalam RAM (Random Access Memory) perangkat. Jadi saat perangkat di wipe, atau di hard reset, data di RAM ini pun dihapus, termasuk kunci untuk dekripsi SD card.

Saat kita pertama melakukan enkripsi SD card, memang diberikan informasi lebih dahulu, kalau SD card tidak akan bisa dibaca di perangkat lain setelah di-enkripsi, atau pada perangkat yang sama kalau di factory reset.

Pengguna disarankan harus melakukan dekripsi SD card terlebih dahulu sebelum melakukan wipe atau factory reset. Karena tingkat enkripsi level tinggi ini, sepertinya anda harus merelakan data di SD card tersebut, tetapi SD card masih bisa digunakan dengan memformat ulang, yang biasanya dibutuhkan bantuan aplikasi lain seperti partition manager.

20. Beberapa anak membagi satu pak kartu. Jika tiap anak mendapatkan 6 kartu maka tersisa 1 kartu. Jika tiap anak mendapatkan 5 kartu maka tersisa 4 kartu. Maka jumlah dari banyak anak dan banyak kartu adalah ...​


Jawaban:

Jumlah anak (x) dan jumlah kartu (y) dapat ditemukan dengan menggunakan dua persamaan:

x * 6 + 1 = y

x * 5 + 4 = y

Menyederhanakan dan memasukkan persamaan kedua ke dalam persamaan pertama, kita memperoleh:

x * 6 + 1 = x * 5 + 4

6x + 1 = 5x + 4x = 3

Jadi, ada 3 anak dan 3 x 6 + 1 = 19 kartu.
