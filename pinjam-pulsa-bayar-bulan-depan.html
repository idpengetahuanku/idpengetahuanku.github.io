Untuk menentukan besarnya pembayaran bulanan pinjaman A sebesar 4 juta rupiah, kita bisa menggunakan rumus:

Pembayaran bulanan = (Pinjaman x Bunga per tahun) / Jumlah bulan

Asumsikan bunga pinjaman adalah sebesar 10% per tahun. Maka, besarnya pembayaran bulanan pinjaman A adalah:

Pembayaran bulanan = (4 juta x 10%) / 18 bulan = 222.22 per bulan

Sedangkan untuk pinjaman B sebesar 3 juta rupiah, besarnya pembayaran bulanannya adalah:

Pembayaran bulanan = (3 juta x 10%) / 18 bulan = 166.67 per bulan

Nilai-nilai ini adalah perkiraan dan dapat berbeda tergantung dari tingkat bunga yang berlaku.

14. pinjaman uang ke bank Rp 2.750.000 dengan bunga 0,75% , akan di bayar selama 8 bulan , berapa ansuran yang harus dibayar setiap bulannya? ​


Besar nya bunga dalam suatu pinjaman di nyatakan dalam % ( persen ) dan disebut dengan suku bunga. Suku bunga per tahun adalah perbandingan antara bunga yg diperoleh dengan modal awal per tahun.

Suku bunga = [tex] \frac {bunga yg diperoleh} {modal awal}[/tex] x 100%

Sedangkan menghitung besar pinjaman untuk beberapa periode adalah sebagai berikut :Mn = Mo ( 1 + nb )

dimana :

Mn = modal akhir

Mo = modal awal  

n = jumlah periode  

b = suku bunga dalam % ( persen ) per periode

Pembahasan  

Mo = Rp 2.750.000

Dianggap bunga 0,75% per bulan, maka :

b = 0,75% x 12 per tahun

b = 9% per tahun

n = 8 bulan

n = [tex] \frac {8} {12}[/tex] tahun

n = [tex] \frac {8 : 4} {12 : 4}[/tex] tahun

n = [tex] \frac {2} {3}[/tex] tahun

Besar angsuran yg harus di bayar selama 8 bulan adalah :

Mn = Mo ( 1 + nb )

Mn = Rp 2.750.000 ( 1 + [tex] \frac {2} {3}[/tex] x 9% )

Mn = Rp 2.750.000 ( 1 + [tex] \frac {2} {3}[/tex] x [tex] \frac {9} {100}[/tex] )

Mn = Rp 2.750.000 ( 1 + [tex] \frac {2} {3 : 3}[/tex] x [tex] \frac {9 : 3} {100}[/tex] )

Mn = Rp 2.750.000 ( 1 + [tex] \frac {2} {1}[/tex] x [tex] \frac {3} {100}[/tex] )

Mn = Rp 2.750.000 ( 1 + [tex] \frac {6} {100}[/tex] )

Mn = Rp 2.750.000 ( 1 + 0,06 )

Mn = Rp 2.750.000 x 1,06

Mn = Rp 2.750.000 x [tex] \frac {106} {100}[/tex]  

Mn = ( Rp 2.750.000 : 100 ) x [tex] \frac {106} {100 : 100}[/tex]  

Mn = Rp 27.500 x 106

Mn = Rp 2.915.000

Besar angsuran yg harus dibayar tiap bulan adalah :

= Rp 2.915.000 : 8

= Rp 364.375

18. Pinjaman cair 7jt Total bayar bpkb 640 per bln Total 11jt520/18bln Pinjaman a 4jt bayaran perbulan harus berapa Pinjaman b 3jt berapa bayaran perbln


Dari informasi yang diberikan, kita tahu bahwa:

Jumlah pinjaman adalah 7 juta rupiah

Total pembayaran untuk pengembalian pinjaman adalah 11.520.000 rupiah, yang dibayar selama 18 bulan.

Ada biaya bulanan sebesar 640 ribu rupiah untuk penggunaan BPKB.

Untuk mengetahui berapa bayaran bulanan untuk masing-masing pinjaman, kita dapat menggunakan rumus angsuran tetap:

Angsuran = (Pinjaman * (Bunga + 1)) / Jumlah Bulan

Namun, dari informasi yang diberikan, tidak ada keterangan mengenai bunga atau tingkat suku bunga yang digunakan oleh pemberi pinjaman. Oleh karena itu, kita akan mengasumsikan bahwa tidak ada bunga yang dikenakan pada kedua pinjaman.

Dengan asumsi bahwa tidak ada bunga yang dikenakan, maka pembayaran bulanan untuk kedua pinjaman dapat dihitung sebagai berikut:

Pinjaman A = 4 juta rupiah

Total pembayaran untuk 18 bulan adalah 11.520.000 rupiah, dengan biaya bulanan sebesar 640 ribu rupiah. Oleh karena itu, pembayaran bulanan untuk pinjaman A dapat dihitung dengan cara berikut:

Pembayaran bulanan = (Total pembayaran - Biaya bulanan * Jumlah bulan) / Jumlah bulan

Pembayaran bulanan = (11.520.000 - 640.000 * 18) / 18

Pembayaran bulanan = 2.240.000 / 18

Pembayaran bulanan = 124.444,44 rupiah

Jadi, pembayaran bulanan untuk pinjaman A sebesar sekitar 124.444,44 rupiah.

Pinjaman B = 3 juta rupiah

Kita dapat menggunakan rumus yang sama untuk menghitung pembayaran bulanan untuk pinjaman B:

Pembayaran bulanan = (Total pembayaran - Biaya bulanan * Jumlah bulan) / Jumlah bulan

Pembayaran bulanan = (11.520.000 - 640.000 * 18) / 18

Pembayaran bulanan = 1.680.000 / 18

Pembayaran bulanan = 93.333,33 rupiah

Jadi, pembayaran bulanan untuk pinjaman B sebesar sekitar 93.333,33 rupiah.
