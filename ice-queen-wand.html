12. Ada berapa Quick Selection & Magic Wand Tool dan sebutkan​


Jawaban:

1. Move Tool

2. Marque Tool

3. Lasso Tool

4. Quick Selection Tool dan Magic Wand Tool

5. Crop dan Slice Tool

Penjelasan:

1. Move Tool

Move Tool berfungsi untuk memindahkan objek baik foto, teks, brush, dan lain-lain. Selain itu, move tool juga dapat digunakan untuk mengatur align pada bagian option box-nya. Cara menggunakannya tinggal menyeret objek yang akan kita pindahkan ke tempat yang diinginkan.

2. Marque Tool

Marque tool berfungsi untuk membuat seleksi. Bila kita mengklik pada tool ini dan menahannya, maka akan muncul jenis dari marque tool. Seperti berikut:

- Reqtangular Marque Tool : Membuat area seleksi berupa segi empat dengan sudut 90 derajat.

- Eliptical Marque Tool : Membuat area seleksi berbentuk lingkaran.

- Single Row Marque Tool : Membuat area seleksi berbentuk garis horizontal.

- Single Column Marque Tool : Membuat area seleksi berbentuk garis vertikal.

3. Lasso Tool

Tool ini memiliki fungsi seperti marque tool, tetapi kita dapat mengaturnya secara manual.

- Lasso Tool : Membuat area seleksi sesuai dengan gerakan cursor mouse.

- Polygonal Lasso Tool : Membuat area seleksi berbentuk persegi atau garis lurus.

- Magnetic Lasso Tool : Membuat area seleksi, tetapi sesuai dengan sudut dari objek.

4. Quick Selection Tool dan Magic Wand Tool

- Quick Selection Tool : Membuat seleksi secara mudah dan cepat.

- Magic Wand Tool : Membuat seleski pada objek yang memiliki warna serupa, toleransinya bisa ditaur pada option box.

5. Crop dan Slice Tool

- Crop Tool : Memotong gambar dengan bentuk persegi atau sejajar.

- Slice Tool : Membuat potongan gambar.

- Slice Select Tool : Memilih potongan gambar bila kita ingin menyeretnya.

Dalam Photoshop yang versinya lebih tinggi dari CS 3, terdapat satu tool lagi pada bagian ini yaitu Perspektif Crop Tool. Berfungsi untuk memotong gambar dan bisa disesuaikan sudutnya, tidak harus 90 derajat.


13. Pengertian toolbox dan magic wand tool adalah


ToolBox adalah kotak berisi objek yang digunakan untuk membuat aplikasi pemrograman dan diletakkan didalam form. Misalnya untuk membuat membuat tulisan, anda dapat mengklik icon label kemudian drag ke dalam form.

Magic Wand Tool - Shortcut (W), digunakan untuk  menyeleksi satu jenis warna (warna solid) pada foto. Alat ini dapat menyeleksi dengan pengaturan nilai tolerance atau nilai cakupan warna, semakin besar nilai tersebut maka semakin luas cakupan warna yang di seleksi.

14. Yg merupakan atribut Magic Wand Tool adalah?


Magic wand tool merupakan salah satu tool yang ada pada program edit gambar seperti Adobe Photoshop atau Adobe Illustrator. Magic wand tool adalah tool yang digunakan untuk menyeleksi piksel berdasarkan warnanya. Cara menggunakan magic wand tool sangat mudah. Kita hanya perlu klik ikon magic wand tool kemudian klik di area yang ingin diseleksi.

Pembahasan

Ikon artibut magic wand tool dapat dilihat seperti pada gambar di lampiran berikut ini. Pada tiap versi Photoshop mungkin bentuknya ada sedikit perubahan-perubahan, namun inti gambarnya tetap sama. Magic wand tool dilambangkan dengan bentuk tongkat sihir yang diujungnya terdapat bintang atau sinar. Untuk lebih jelasnya, berikut ini saya uraikan contoh penggunaan magic wand tool dan langkah-langkah teknisnya:

Buka Adobe Photoshop.Masukkan gambar yang akan diedit.Klik ikon magic wand tool yang ada pada toolbox.Klik pada bagian gambar yang memiliki warna sama.Bagian yang memiliki warna yang sama tersebut akan terseleksi dan siap diedit sesuai keinginan kita.

Pada magic wand tool terdapat pilihan tolerance. Pengaturan angka tolerance akan menunjukkan toleransi perbedaan warna yang hendak diseleksi. Semakin besar angka tolerance maka semakin besar pula kemungkinan perbedaan warna yang ikut terseleksi. Begitu pula sebaliknya semakin kecil angka tolerance maka semakin kecil pula kemungkinan warna yang berbeda akan terseleksi.

17. Usia queen 5/8 dari usia saffa. Usia saffa 3 tahun lebih tua dari usia queen. Berapa usia queen


Jawaban:

q = queen

s = saffa

q = 5/8 s

8q = 5s

s = 3 + q

8q = 5(3 + q)

8q = 15 + 5q

8q - 5q = 15

3q = 15

q = 15/3

q = 5

s = 3 + q => 3 + 5 = 8
