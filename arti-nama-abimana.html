Misalkan kedua bilangan tersebut adalah a dan b

a² + b² = 89

a . b = -40

i) a² + b² = 89

(a + b)² - 2ab = 89

(a + b)² - 2(-40) = 89

(a + b)² + 80 = 89

(a + b)² = 89 - 80 = 9

a + b = ± 3

Kemungkinannya adalah a + b = 3 atau a + b = -3

ii) a² + b² = 89

(a - b)² + 2ab = 89

(a - b)² + 2(-40) = 89

(a - b)² - 80 = 89

(a - b)² = 89 + 80 = 169

a - b = ± 13

Kemungkinannya adalah a - b = 13 dan a - b = -13

Kemungkinan nilai a dan b:

1. Kemungkinan 1:

a + b = 3

a - b = 13

------------- -

2b = -10

b = -5

a - 5 = 3 --> a = 8

Maka (a, b) = (8, -5)

2. Kemungkinan 2:

a + b = 3

a - b = -13

-------------- -

2b = 16

b = 8

a + 8 = 3

a = 3 - 8 = -5

Maka (a, b) = (-5, 8)

3. Kemungkinan 3:

a + b = -3

a - b = 13

------------- -

2b = -16

b = -8

a - 8 = -3

a = -3 + 8 = 5

Maka (a, b) = (5, -8)

4. Kemungkinan 4:

b = 5
