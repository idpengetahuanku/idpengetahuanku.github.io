Menyebutkan nama-nama bulan dalam bahasa Jepang itu mudah. Hanya perlu menghafal angka 1-12 dalam bahasa Jepang.

Pola kalimat :

[Bulan ke... ]  + 月 (gatsu)

・September, bulan ke 9. Jadi penulisannya adalah 9月 (dibaca : ku-gatsu)

2. nama nama bulan dalam bahasa Jepang ​


Januari = 一月 (Ichi gatsu)Februari = 二月 (Ni gatsu)Maret = 三月 (San gatsu)April = 四月 (Shi gatsu)Mei = 五月 (Go gatsu)Juni = 六月 (Roku gatsu)Juli =  七月 (Shichi gatsu)Agustus = 八月 (Hachi gatsu)September = 九月 (Ku gatsu)Oktober = 十月 (Juu gatsu)November = 十一月 (Juu ichi gatsu)Desember = 十二月 (Juu ni gatsu)

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan

Bulan dalam Bahasa Jepang adalah 月 (Gatsu). Kanji 月 bisa juga dibaca 'Tsuki'. Perbedaannya kalau dalam Bahasa Inggris yaitu, Gatsu berarti "Month", sedangkan Tsuki berarti "Moon". Penyebutan bulan dalam Bahasa Jepang caranya sangat mudah. Kalian hanya perlu menghafal angka 1-12 saja. Pola dalam penyebutan bulan adalah sebagai berikut.

=> [Bilangan Bulan] + 月

=> [Bilangan Bulan] + gatsu

--

Contohnya, bulan Januari. Bulan Januari adalah bulan ke-1, maka penulisannya dalam Bahasa Jepang adalah 一月 (Ichi gatsu). Contoh lainnya bulan Maret, yaitu bulan ke-3. Maka penulisannya dalam Bahasa Jepang adalah 三月 (San gatsu).

--

Angka-angka yang digunakan dalam penyebutan bulan adalah sebagai berikut.

Ichi = 1                Shichi = 7

Ni = 2                 Hachi = 8

San = 3               Ku = 9

Shi = 4                Juu = 10

Go = 5                Juu ichi = 11

Roku = 6            Juu ni = 12

*Dalam penyebutan bulan, angka 4 dibaca Shi, angka 7 dibaca Shichi, dan angka 9 dibaca Ku.

setelah belajar tentang kalender dalam bahasa jepang. hal apa yang membedakan dengan penyebutan kalender di indonesia?

bulan Jepang ditulis dengan urutan angka 1 mencapai 12,sedangkan di indonesia tidak ditulis dengan urutan angka 1 mencapai 12, bulan di indonesia mirip dengan bahasa inggris (menggunakan istilah  peradaban Romawi)

bulan dalam nihongo itu gatsu (月)

jadi supaya mudah menghapalnya,

bulan ke 1 adalah januari, bahasa jepang 1 adalah ichi. jadi januari dalam nihongo adalah ichi gatsu (artinya bulan ke 1)

angka + gatsu = bulan

angka dalam nihongo

1= ichi  = いち = 一

2= ni  = に = 二

3= san  = さん = 三

4= shi/ yon  = し、よん = 四

5= go  =ご =五

6=roku  = ろく = 六

7=shichi / nana  = しち、なな = 七

8= hachi  = はち = 八

9=kyuu / ku  =  く、きゅう = 九

10=juu = じゅう = 十
