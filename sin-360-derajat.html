1. sin (x + 25) derajat = sin 40 derajat pada derajat derajat


sin ( x + 25 )° = sin 40°

x + 25° = 40° + k. 360°

x = 15° + k.360°

k = 0. x1 = 15°

k = 1. x3 = 375° ( TM )

x + 25° = ( 180° - 40° ) + k.360°

x = 115° + k.360°

k = 0. x2 = 115°

k = 1. x4 = 475° ( TM )

HP = { 15°, 115° }


2. Nilai Sin 0 ≠ Sin 360 derajat


Nilai Sin 0 ≠ Sin 360 derajat.

Jawaban: Sin 0 = Sin 360 derajat = 0.

PEMBAHASAN:

Sinus:

Sinus suatu sudut didefinisikan sebagai perbandingan panjang sisi di depan sudut dengan sisi miring.

Sin = sisi depan ÷ sisi miring.

Cosinus:

Cosinus suatu sudut didefinisikan sebagai perbandingan panjang sisi di samping sudut dengan sisi miring.

Cos = sisi samping ÷ sisi miring.

Tangen:

Tangen suatu sudut didefinisikan sebagai perbandingan panjang sisi di depan sudut dengan sisi di samping sudut.

Tangen = sisi depan ÷ sisi samping.

8. himpunan penyelesaian dari sin x = sin 70 derajat, 0 derajat kurang dari x kurang dari 360 derajat adalah​


Himpunan penyelesaiannya adalah HP = {70°, 110°}. Cara mendapatkan silakan disimak dalam pembahasan.

Pembahasan

Persamaan trigonometri adalah suatu persamaan yang di dalamnya terkandung trigonometri seperti sinus, cosinus, dan tangen. Berikut untuk menyelesaikan persamaan trigonometri

sin x = sin θ

x = θ + k . 360°

x = (180° - θ) + k . 360°

cos x = cos θ

x = θ + k . 360°

x = -θ + k . 360°

tan x = tan θ

x = θ + k . 180°

Oke, langsung saja kita akan membahas soalnya.

Penyelesaian

Sin x  = sin 70°

Maka, sesuai penjelasan di  atas menjadi

x = 70° + k . 360°

k = 0 ⇒ x = 70° + 0 . 360° = 70°

k = 1 ⇒ x = 70° + 1 . 360° = 70° + 360° = 430° (Tidak termasuk karena sudah lebih dari 360°)

x = (180° - 70°) + k . 360°

x = 110° + k . 360°

k = 0 ⇒ x = 110° + 0 . 360° = 110°

k = 1 ⇒ x = 110° + 1 . 360° = 110° + 360° = 470° (Tidak termasuk karena sudah lebih dari 360°)

HP = {70°, 110°}
