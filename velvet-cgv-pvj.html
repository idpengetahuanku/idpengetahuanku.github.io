r = d ÷ 2

 = 14 ÷ 2

 = 7 cm

V = [tex]\frac{1}{2}[/tex] × [tex]\frac{4}{3}[/tex] × [tex]\pi[/tex] × r × r × r

17. Nada membeli dessert box yang dijual oleh temannya di Instagram. Harga 1 box tiramisu sama dengan harga 2 box red velvet. Harga 3 box tiramisu dan 2 box red velvet Rp480.000. Apabila Nada ingin membeli 2 box tiramisu dan 3 box red velvet, berapa harganya?NB : tidak ada diskon atau harga teman. Bisnis tetap bisnis guys.​


Nada membeli dessert box yang dijual oleh temannya di Instagram. Harga 1 box tiramisu sama dengan harga 2 box red velvet. Harga 3 box tiramisu dan 2 box red velvet Rp480.000. Apabila Nada ingin membeli 2 box tiramisu dan 3 box red velvet, harganya adalah [tex]\boxed{\sf{Rp420.000}}.[/tex]

ㅤ

ㅤ

PEMBAHASAN

SPLDV adalah singkatan dari Sistem Persamaan Linier Dua Variabel. SPLDV adalah persamaan matematika yang terdiri dari dua variabel berbeda dan lebih dari satu persamaan. Adapun bentuk umum dari SPLDV adalah:

[tex]\boxed{\boxed{\left\{\begin{array}{c} \sf{ax+by=c}\\\sf{px+qy=r}\end{array}\right.}}[/tex]

ㅤ

Cara penyelesaian SPLDV diantaranya:

Metode grafikMetode eliminasiMetode substitusiMetode campuranMetode matriksDll

ㅤ

ㅤ

Diketahui:

Harga 1 box tiramisu sama dengan harga 2 box red velvet

Harga 3 box tiramisu dan 2 box red velvet Rp480.000

ㅤ

Ditanyakan:

Harga 2 box tiramisu dan 3 box red velvet adalah …

ㅤ

Jawab:

Pertama-tama ubahlah kalimat diatas menjadi kalimat matematika agar lebih mudah untuk diselesaiakan.

ㅤ

Misalkan:

x = Harga box tiramisu

y = Harga box red velvet

ㅤ

Maka:

Harga 1 box tiramisu sama dengan harga 2 box red velvet dapat ditulis menjadi [tex]\boxed{\sf{x=2y}}.[/tex]Harga 3 box tiramisu dan 2 box red velvet Rp480.000 dapat ditulis menjadi [tex]\boxed{\sf{3x+2y=480.000}}.[/tex]Harga 2 box tiramisu dan 3 box red velvet adalah … dapat ditulis menjadi [tex]\boxed{\sf{2x+3y=...}}.[/tex]

ㅤ

Diperoleh bentuk umum SPLDV:

[tex]\left\{\begin{array}{c}\sf{\:\:\:x =2y}\\ \sf{3x+2y=480.00}\end{array}\right.[/tex]

ㅤ

Untuk penyelesaiannya gunakan metode substitusi agar lebih mudah. Ada dua cara yang bisa dipilih untuk menyelesaikan permasalahan diatas.

CARA I: Cari nilai x dan nilai y.

[tex]\begin{array}{cl}\sf{\:\:\:x =2y}&...\:\sf{(i)}\\\sf{3x+2y=480.00}&...\:\sf{(ii)}\end{array}[/tex]

ㅤ

Substitusikan persamaan (i) ke persamaan (ii).

[tex]\begin{array}{rll}\sf{3x+2y}&=&\sf{480.000}\\\sf{3x+\:\:x}&=&\sf{480.000}\\\sf{4x}&=&\sf{480.000}\\\sf{x}&=&\sf{120.000}\end{array}[/tex]

ㅤ

Substitusikan nilai x = 120.000 ke persamaan (i)

[tex]\begin{array}{lll}\sf{\:\:x}&=&\sf{2y}\\\sf{2y}&=&\sf{x}\\\sf{2y}&=&\sf{120.000}\\\sf{\:\:y}&=&\sf{60.000}\end{array}[/tex]

ㅤ

Sehingga harga 2 box tiramisu dan 3 box red velvetnya adalah

[tex]\begin{array}{lll}\sf{2x+3y}&=&\sf{2(120.000)+3(60.000)}\\&=&\sf{240.000+180.000}\\&=&\sf{420.000}\end{array}[/tex]

ㅤ

Jadi harga 2 box tiramisu dan 3 box red velvetnya adalah [tex]\boxed{\sf{Rp420.000}}.[/tex]

ㅤ

CARA II: Cukup cari nilai y.

[tex]\begin{array}{cl}\sf{\:\:\:x =2y}&...\:\sf{(i)}\\\sf{3x+2y=480.00}&...\:\sf{(ii)}\end{array}[/tex]

ㅤ

Substitusikan persamaan (i) ke persamaan (ii).

[tex]\begin{array}{rll}\sf{3x+2y}&=&\sf{480.000}\\\sf{3(2y)+2y}&=&\sf{480.000}\\\sf{6y+2y}&=&\sf{480.000}\\\sf{8y}&=&\sf{480.000}\\\sf{y}&=&\sf{60.000}\end{array}[/tex]

ㅤ

Sehingga harga 2 box tiramisu dan 3 box red velvetnya adalah

[tex]\begin{array}{lll}\sf{2x+3y}&=&\sf{2(2y)+3y}\\&=&\sf{4y+3y}\\&=&\sf{7y}\\&=&\sf{7(60.000)}\\&=&\sf{42.000}\end{array}[/tex]

ㅤ

Jadi harga 2 box tiramisu dan 3 box red velvetnya adalah [tex]\boxed{\sf{Rp420.000}}.[/tex]

19. Hari ini ali dan khai berulang tahun. Karena sedang pandemi Xabiru dan Chava merayakan dengan membagikan kue untuk teman-temannya. Xabiru membeli 12 kue red velvet dan 15 kotak kue cokelat dengan harga Rp. 240.000,00. Ditempat yang sama, Chava membeli 7 kue red velvet dan 12 kue cokelat dengan harga Rp. 166.000,00. Karena masih ada temannya yang belum kebagian, Xabiru membeli lagi 8 kue red velvet dan 5 kue cokelat dengan harga Rp 120.000,00 dan Chava membeli lagi 5 kue red velvet dan 3 kue cokelat dengan harga Rp 74.000,00. Jika aji membeli 3 kue red velvet dan 2 kue cokelat dan membayar dengan uang 1 lembar Rp 100.000,00, berapakah uang kembalian yang akan diterima oleh aji? Selesaikanlah permasalahan diatas dengan metode campuran (subtitusi dan eliminasi)​


Jawaban:

Uang kembalian = 54rb

Penjelasan dengan langkah-langkah:

Kita ketahui sebagai berikut,

12 Red Velvet + 15 Coklat = 240rb7 Red Velvet + 12 Coklat = 166rb8 Red Velvet + 5 Coklat = 120rb5 Red Velvet + 3 Coklat = 74rb

Mari kita sepakati bahwa,

Red Velvet = RKue Coklat = C

Untuk persamaan ke-3, dapat kita ubah,

8R + 5C = 120rb

3(8R + 5C) = 3(120 rb)

24R + 15C = 360rb

Eliminasi C pd persamaan 1,

24R + 15C = 360rb

12R + 15C = 240rb

12R = 120rb

R = 10rb

Dengan substitusi, kita dapatkan

5R + 3C = 74rb

5(10rb) + 3C = 74rb

50rb + 3C = 74rb

3C = 24rb

C = 8rb

Uang kembalian,

= 100rb - (3R + 2C)

= 100rb - 3R - 2C

= 100rb - 3(10rb) - 2(8rb)

= 100rb - 30rb - 16rb

= 54rb
