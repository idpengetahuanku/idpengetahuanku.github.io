Misal umur mereka sekarang.

Berlian = B

Naura = N

B = N + 5

6 tahun akan datang;

B + 6

N + 6

( B + 6 ) + ( N + 6 ) = 51

B + N + 12 = 51

N + 5 + N + 12 = 51

2N = 51 - 5 - 12

2N = 34

N = 34/2

N = 17

B = 17 + 5

B = 22

jadi umur Berlian sekarang 22 tahun.   ( D )
