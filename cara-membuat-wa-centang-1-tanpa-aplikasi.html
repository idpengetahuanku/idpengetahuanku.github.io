7. Jika tombol ini di centang akan membuat karakter dapat bergerak berulang ulang


Menu perintah untuk menambahkan karakter pada software animasi pivot adalah menu Create Figure Type, nah menu tersebut berada di dalam menu FIle yang ada di Tool bar. Cara mengaksesnya adalah dengan mengeklik menu File tersebut, kemudian pilih menu Create Figur Type yang berada pada baris ke - 6.
Pembahasan
Selain menu Create Figur Type pada menu utaa File, dalam software animasi Pivot juga terdapat beberapa menu - menu lain yaitu :
New digunakan untuk membuat file baru
Open Animation digunakan untuk membuka file yang disimpan
Save Animation digunakan untuk menyimpan file yang sudah jadi. Untuk menyimpan file dengan nama yang berbeda, gunakan Save Animation\ as Animation Export
Load Background berfungsi untuk menambahkan background
Load Sprite Image berfungsi untuk menambahkan karakter dari luar. Karakter tersebut dapat berupa gambar
Load Figure Type berfungsi untuk menambahkan karakter.
Create Figure Type digunakan ketika akan membuat karakter baru.
Exit digunakan untuk keluar dan menutup aplikasi pivot.
