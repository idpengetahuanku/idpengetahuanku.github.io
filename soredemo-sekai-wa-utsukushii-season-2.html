2. arti mada kono sekai wa​

Pembahasan

Berikut terjemahan beserta juga dengan penjelasan kata-perkatanya.

Mada「まだ」: memiliki 2 arti yakni belum dan masih. Tergolong jenis na-keiyoushi (kata sifat -na) dengan kelompok yang ke-ll. Namun kata mada bisa tergolong menjadi kata keterangan.

Kono「この」: kata tunjuk dan jika dalam bahasa Indonesia yang berarti ini. Menunjuk ke benda yang dekat jaraknya.

3. arti "mada kono sekai wa" apa?​


mada kono sekai wa dalam bahasa Indonesia arti kalimat tersebut adalah (sepertinya) dunia ini masih ....

Pembahasan

まだこの世界は ....

Mada kono sekai wa ...

(sepertinya) dunia ini masih ....

_______

–› Mada「 まだ 」

Memiliki 2 arti dalam bahasa Indonesia yaitu adalah masih atau belum (tergantung konteks/penggunaan).

Secara umum, jika kata まだ (mada) tersebut bertemu dengan kata kerja maka artinya dapat diartikan belum.

Bentuk kata まだ (mada) tersebut merupakan bentuk kata sifat yang termasuk golongan na-keiyoushi (akhiran -na) ll.

–› Kono「 この 」

Artinya yaitu ini. Jika ingin merujuk atau menunjukkan suatu subjek, benda/objek yang jaraknya tidak jauh alias berada di dekat diri kita maka kata tunjuk yang cocok digunakan untuk hal ini adalah この (kono).

–› Sekai「 世界 • せかい 」

Kata tersebut terdapat 2 arti yaitu bisa diartikan semesta ataupun dunia.

Jika kata sekai memiliki awalan huruf i, maka artinya menjadi lain yaitu 異世界 (isekai) yang artinya adalah dunia lain/dunia berbeda.

–› Wa「 は 」

Penggunaannya sekedar sebagai kata bantu yang penulisannya ditulis dengan huruf hiragana, kata bantu yang dimaksud yakni sebagai penanda dalam kalimat. は (wa) termasuk kata bantu ataupun juga merupakan jenis partikel.

'Mada kono sekai wa' dalam Bahasa Indonesia artinya adalah "Dunia ini masih..."

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :

Berikut pembahasan per-kata.

MADA「まだ」

Bisa berarti "Masih" atau "Belum".

KONO「この」

Berarti "Ini". Merupakan kata tunjuk untuk menunjukkan sesuatu yang dekat dengan pembicara. Kata ini tidak dapat berdiri sendiri, dan harus menerangkan sesuatu setelahnya. Contohnya pada kalimat ini, yang diterangkan adalah 'Sekai' (dunia). Kalau diartikan menjadi "Dunia ini".

SEKAI「世界・せかい」

Berarti "Dunia".

WA「は」

Tidak ada arti. Merupakan partikel yang memiliki fungsi untuk menandakan subjek atau topik dari sebuah kalimat. Dalam beberapa situasi, partikel ini dapat diartikan "Adalah". Penulisan partikel ini dengan huruf Ha, tapi dibaca Wa.

- - - - - -

Kalimat 'Mada kono sekai wa' ini kemungkinan merupakan potongan lirik dari lagu Sparkle oleh RADWIMPS. Kalau berdasarkan lirik lagunya, kalimat lengkapnya seharusnya 'Mada kono sekai wa boku wo kainarashιtetai mitai da'. Artinya adalah "Sepertinya dunia ini masih ingin mengendalikanku".

Berikut pembahasan setiap kata yang tersisa di kalimat lengkapnya.

BOKU「僕・ぼく」

Berarti "Aku". Kata ini merupakan kata ganti orang pertama tunggal yang biasa dipakai di situasi informal, tapi masih terkesan sopan. Umumnya digunakan laki-laki saja.

WO / O「を」

Tidak ada arti. Merupakan partikel untuk menandakan objek atau kata benda. Biasa diletakkan di antara kata benda dan kata kerja. Pada kalimat ini, yang ditandai sebagai kata benda adalah 'Boku' (saya). Kalau dalam Hiragana ditulis dengan huruf Wo, tapi dalam Romaji bisa ditulis dengan huruf Wo maupun O.

KAINARASHlTETAI「飼いならしてたい」

Berarti "(Sedang) ingin menjinakkan". Kata kerja pada bentuk ini menunjukkan suatu keadaan yang masih ingin berlanjut. Kadang diartikan sebagai "Sedang ingin...". Kalau melihat konteks kalimat, kata ini sebenarnya lebih tepat kalau diartikan sebagai "(Sedang) ingin mengendalikan".

MITAI「みたい」

Berarti "Kelihatannya", "Sepertinya", "Seperti", dsb. Kata ini merupakan kata sifat-i, atau kata sifat yang terdapat tambahan huruf i di akhir. Kata ini diletakkan setelah predikat, untuk menyatakan suatu kemiripan pada predikatnya.

DA「だ」

Tidak ada arti. Fungsinya sebagai penegas kalimat. Mirip seperti 'Desu', tapi 'Da' tidak memperhalus kalimatnya.

9. apa perbedaan きれいい ( kireii ) dengan 美しい ( utsukushii )?​


Jawaban :

1). 綺麗  /  きれい  /  Kirei

Bisa berarti "Indah", "Cantik", "Bersih", "Rapih", dll.Biasa digunakan sebagai ungkapan dalam bentuk lisan.Termasuk kata sifat-na.Umum digunakan.Maknanya tidak terlalu emosional.

2). 美しい  /  うつくしい  /  Utsukushii

Bisa berarti "Indah", "Cantik", "Anggun", dll.Biasa digunakan sebagai ungkapan dalam bentuk tulis.Termasuk kata sifat-i.Tidak umum digunakan.Maknanya lebih emosional.

Pembahasan :

Kata sifat (KS) dalam Bahasa Jepang disebut dengan Keiyoushi. Kata sifat Bahasa Jepang dibagi menjadi dua kelompok. Pengelompokan kata sifat ini nantinya akan berpengaruh saat akan dikonjugasikan. Kedua kelompok kata sifat Bahasa Jepang yakni :

Kata sifat-i / i-keiyoushi = Kata sifat yang berakhiran "i". Huruf "i" di sini merupakan tambahan huruf, bukan bagian dari kata asli nya.Kata sifat-na / na-keiyoushi = Kata sifat yang berakhiran "na". Tapi akhiran "na" ini hanya digunakan saat kata sifat nya menerangkan sesuatu saja. Jika tidak, maka tidak perlu menambahkan "na".

= = = = = = = = = = = = = =

Contoh-contoh kata sifat-i

大きい (Ookii) = Besar小さい (Chiisai) = Kecil長い (Nagai) = Panjang短い (Mijikai) = Pendek高い (Takai) = Tinggi低い (Hikui) = Rendah美味しい (Oishii) = Enak不味い (Mazui) = Tidak enak明るい (Akarui) = Terang暗い (Kurai) = Gelap美しい (Utsukushii) = Indah, Cantik, dll

= = = = = = = = = = = = = =

Contoh-contoh kata sifat-na

静か (Shizuka) = Sepi, Tenang, dll賑やか (Nigiyaka) = Ramai元気 (Genki) = Sehat, Bersemangat, dll綺麗 (Kirei) = Indah, Cantik, dll簡単 (Kantan) = Mudah好き (Suki) = Suka上手 (Jouzu) = Pandai下手 (Heta) = Payahハンサム (Hansamu) = Tampan親切 (Shinsetsu) = Baik hati真面目 (Majime) = Serius, Rajin, dll

= = = = = = = = = = = = = =

Kenapa kata 'Kirei' termasuk KS-na?

Pada kata 'Kirei', huruf "i" di akhir nya merupakan bagian dari kata asli nya. Huruf "i" di sini merupakan bagian dari Kanji 麗 (Rei). Seperti yang sudah saya jelaskan sebelumnya, kata sifat-i akhiran huruf "i" nya merupakan tambahan dan bukan bagian dari kata asli nya.

17. apa perbedaan (utsukushii) 美しい/ (kirei) 綺麗 dengan 素敵 (suteki)?​


Silakan simak di pembahasan ya.

Pembahasan

Utsukushii「美しい • うつくし」Artinya : cantik, indah.

Penggunaannya ini umumnya merujuk kepada hal yang indah maupun cantik namun dengan suatu hal yang anggun, memiliki kesan yang elegan, dan adanya kesan yang tidak biasa dalam perasaan. Seperti hal-hal yang bagi kita enak dan nyaman untuk dilihat maupun dipandang, hal yang dapat menarik perhatian, contohnya seperti suara, warna, atau sosok/karakter dan wajah.

Untuk golongan kata utsukushii ini tergolong jenis kata sifat atau i-keiyoushi (kt. sifat akhiran -i) dengan golongan yang ke-l.

________

Kirei「綺麗 • きれい」Artinya : bagus, indah, cantik.

Untuk penggunaannya sendiri berbeda dari yang sebelumnya, yaitu yang dimana kata kirei ini lebih merujuk hal yang memiliki kesan nyaman dan enak untuk dipandang atau dilihat namun tidak ada kesan yang dapat melibatkan perasaan.

Sedangkan untuk golongan kata dari kata kirei adalah tergolong na-keiyoushi atau yang berarti kata sifat akhiran -na dengan kelompok ke-ll.

________

Suteki「素敵 • すてき」Artinya : menarik, bagus, baik, indah.

Penggunaan kata suteki umumnya merujuk ke suatu benda yang memiliki kesan yang bagus, menarik, atau sesuatu hal yang dapat menarik perhatian, dan hal yang menggambarkan karakter atau sosok yang bagus dan hebat.

Dapat juga merujuk ke hal-hal yang melibatkan perasaan dengan perasaan yang baik atau bagus. Kata suteki tergolong na-keiyoushi (kt. sifat akhiran -na), termasuk kelompok ke-ll.

20. Penulisan bahasa Jepang kata sekai untuk benar dan sekai untuk dunia


Kata 'benar' bahasa Jepangnya adalah : 正解 (seikai).Kata 'dunia' bahasa Jepangnya adalah : 世界 (sekai).

Pembahasan

Perbedaan antara 正解 dan 世界

正解 (seikai)

Jika diartikan, maknanya adalah benar. Walau jika ditulis dalam huruf romajinya adalah seikai, namun untuk penyebutannya dibaca sebagai seekai (huruf 'e' dibaca panjang).

Kata 正解 ditulis dengan menggunakan huruf kanji, jika diubah menjadi huruf hiragana maka adalah せいかい

世界 (sekai)

Memiliki arti yaitu dunia atau semesta. Untuk penyebutannya sesuai dengan huruf romaji yang tertera.

Kata 世界 juga ditulis sebagai huruf kanji. Namun jika kita ubah bentuk kanji ke hiragana maka adalah せかい
