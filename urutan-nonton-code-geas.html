13. Jelaskan pengertian kode angka berurutan (sequence number code)!


Pengertian kode angka berurutan yaitu akun diberi nomor secara berurutan. Biasanya dimulai dengan 1 dan 100.
Contoh nya yaitu :
Kas misalnya diberi nomor 1
Piutang usaha diberi nomor 2
Perlengkapan diberi nomor 3
Mesin diberi nomor 11
Akumulasi penyusutan diberi nomor 12

14. Buat algoritma pseudo code untuk mengurutkan angka 1 sampai dengan 8!


// ConsoleApplication2.cpp : Defines the entry point for the console application.

//

#include "stdafx.h"

#include <iostream>

using namespace std;

int main()

{

int n=8, i, j, temp;

int arr[8] = {8,5,6,3,4,7,2,1};

cout << "Nilai Awal = 8 5 6 3 4 7 2 1 \n\nSetelah Di Sorting : ";

for (i = 0; i<(8 - 1); i++)

{

 for (j = 0; j<(n - i - 1); j++)

 {

  if (arr[j]>arr[j + 1])

  {

   temp = arr[j];

   arr[j] = arr[j + 1];

   arr[j + 1] = temp;

  }

 }

}

for (i = 0; i<8; i++)

{

 cout << arr[i] << " ";

}

system("pause");

   return 0;

}
